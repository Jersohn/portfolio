Weather App

Application Météo Janvier 2023

Application météo interactive utilisant une API pour afficher les conditions météorologiques en temps réel dans le monde entier. L'application offre une interface intuitive et éveloppée en pur JavaScript sans frameworks.

Fonctionnalités clés :

  • Recherche par ville ou géolocalisation automatique
  • Interface responsive adaptée à tous les appareils
